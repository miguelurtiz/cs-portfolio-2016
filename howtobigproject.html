                    function getMousePos(canvas, evt) {
var rect = canvas.getBoundingClientRect();
return {
  x: evt.clientX - rect.left,
  y: evt.clientY - rect.top
};
}

With this code, I am able to find the position coordinates of the mouse when it's upon the canvas. The 'X' value indicates when the mouse moves either left or right. The 'Y' value indicates when the mouse moves either up or down.

                    canvas.addEventListener('mousedown', function(evt){

    context.fillStyle = "black";

    var mousePos = getMousePos(canvas, evt)
    var x = (mousePos.x - (mousePos.x % 20))
    var y = (mousePos.y - (mousePos.y % 20))
    context.fillRect(x, y, 20, 20);
}, false)

This coding is very important. This code is used where when you click your mouse on a pixel space that is on the canvas, it fills that pixel space with a black color. The lower portion of the code has a special unit where it identifies: the X axis, the Y axis, the width, and the height (X,Y,W,H). By making both the width and height values 20 (X,Y,20,20), I am able to adjust the size of each and every pixel into 20x20.
