The Navigation Bar

The Navigational Bar above the screen is a tool used to navigate through different pages of the website. However, in order to create the Navigational Bar, I had to use both HTML and CSS to construct the bar above each webpage. Firstly, we'll talk about the...

HTML

Coding with HTML would help create the specific elements such as titles, text, buttons, images, and videos, that would be present on any webpage in the world wide web.

Firstly, we need to create a list in the HTML file by placing a "ul" tag in the "body". We would also need to place a "ul id", this would title the list itself in order for the CSS to recognize the name of the list. You would place your title within the green qutation marks, so we'll title it, "navbar".





Next, we need to create list elements in our list, to do so, we need to place an "li" tag followed by an "a href" tag to indicate the name of the list element and the link that it send you to. We would have to place the HTML file name of the webpage link in the green quotation marks (index.html), and the title of the list element in the angle brackets (About Me).



We would also have to do the same process for the Hardware webpage list element, however, we have to take away the "a href" tag because there would be no point of placing a link to a webpage that we are already on. By doing this, the word "Hardware" would be visable on the navigational bar, but it wouldn't be clickable, giving the illusion that it is already selected.



For the following list elements that I have not created a webpage link for yet, I must place a hashtag mark as a substitute for the webpage links within the "a href" tags. This would make the title of the webpages on the navigational bar visable, but unclickable, due to the webpages not existing yet. Lastly we now have to close the list with a "/ul" tag to let the HTML file know that we are finished creating list elements.


By coding the entire list on the HTML, we have created the Navigational Bar above. However, it looks to be blank and boring. To fix up the position and the look of the navigational bar, we have to code using...

CSS

Coding with CSS would help change te look and the positions of the elements created with HTML. Rather from using tags, we would mostly use properties to identify what we want to change.

Firstly, before we even think about doing anything to the CSS file, we must first link the CSS file to the HTML file by using a "link rel" tag. At the same time, we would have to pace the name of the CSS file in between the green quotation marks near the "href" tag. This would give the information about the CSS to the HTML file. Without it, the CSS file would be useless.





Next, we have to change the looks of the Navigational Bar titled "navbar" by creating a "#navbar" tag on the CSS file. This would custimize the look and the the position of the border behind the list elements.






In order to change the position of the navbar, we have to type either "left", "center", or "right" onto the "text-align" property. This would position the navigational bar to the center of the webpage.






Next, we have to select a color for the navigational bar from a hex color palette. To select a specific color from a hex color palette, we would have to use a website to produce a 6-digit code for the color we would want to use (#4da6ff). Once we find the color we want, we would have to paste it near the "background-color" property. This would automatically change the navbar to a nice blue color.



Then, we would have to change the style of the border surrounding the text of list elements. to do this, we would have to type in "ridge" within the "border-style" property. Doing this would create a ridged looking outline for the border.






Lastly, in order to select the size of the border itself, we would need to add a "padding" propery to the "#navbar" tag. This would increase or decrease the size of your navbar box, depending on the number you've selected. We're almost done, now we have to code a new set of properties in order to modify the look and positions for the clickable list elements.



Firstly, in order for the CSS file to recognize that we're selecting the list elements, we need to add a new "#navbar > li" tag. This would let the CSS file know that we want to change the looks and position of the list elements.






Next, we would have to add a "background-color" property and select the same hex color code as the one we did previously.







Then, we would have to add a new "display" property and type "inline-block" near it. This will help change the position of the list elements from being stacked vertically to being rowed horizontally.






Eventually, we would have to create another "padding", in order to change the size of the boxes to fit the list element in.





Lastly, we would have to create a border surrounding our button, so in order to to that, we would need to create another "border-style" property. However, instead of using the same border design as the previous one, we're going to type in "outset", to give the navbar some variety.

The Format

The format of my hardware page is as simple as it goes, first the title, then the image, the text, the history, a video, then lastly, the border line.


I'm going to give step-by-step instructions on how to create a website with my style and formating.


Firstly, in order to create the title, I have to use a title, I need to add the "font size" tag to adjust the size of the title font. Then I would have to type in what I want the title to say in between the "h2" tags.







Secondly, In order to create an image, we would need to add an "img src" tag in order to place an image source in between the qutation marks. with the "style" tag, we can change the height and the width of the image.







Eventually, we would need to add text in order to describe what we are talking about on the hardware page. To insert text, we need to add a "font size" tag to create and determine the size of the font we would be using. Once you select the size you want, you would need to create a paragraph tag and type in the amount of text you would need to put in to describe what to say. (I put "text" between the paragraph tags to show an example.)

Now, since we want to include the history of each piece of hardware, we would first move to another line below and do the exact same steps as we did with making simple text. However, we would need to place the word "History: " in the paragraph tag. Once done, just write about the hisory after the phrase "History: ".





Next, We would need to add a youtube video talking about the hardware item. To do so, we would have to make an "iframe" tage and add the height and the width of the video frame. Then you would have to insert the youtube video link within the "src" tag. It's also very important to surround the "iframe" tag with "center" tags, that way, the video will be placed in the center of the webpage.



Lastly, and the most easy step to do, is insert an "hr" tag in between segments to create a white lined border to separate segments and make the page look more organized.
