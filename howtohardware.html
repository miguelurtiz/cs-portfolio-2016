How To: Hardware Page

Each "How To" Page will give me the chance to explain the different coding methods that I use when creating this website about me and my computer scinece experiences.

The Background

The background is created using a cool looking dotted background that I found on Google Images. Next, I use this particular code above and insert that image's address, the one in green text. Then, I arrange the photo to fill up the entire screen and to stay fixed so that it doesn't repeat by entering "no-repeat center center fixed".

The Navigation Bar

The Navigational Bar above every single page on this website is programmed by typing the names of those pages and connecting those names with links to the next page. By connecting the name "howtohardware.html" to "How To", I am able to click the word "How To" and be able to go to a page named "howtohardware.html".

The Format

The Format of each portion of the Hardware Page goes in this order: Title, Picture, Text about Picture, Video about Hardware, Line Border, Repeat. I know that most of the text doesn't fit in the photo on the left, but this will have to do. Firstly, In order to write the title, we write the title name under "h2" and it will specifically enlarge the text to make the title stand out more. Next, we need to add an image, so I got an image from Google Images, copy and pasted the image address, and fixed the image size. Then, I have to add some text talking about the image, so I input some text under "p", which stands for paragraph. Next, I have to add a video about the topic, so I embed a YouTube video and paste it on the coding titled "iframe". lastly, I want to border the different hardwares, so I have to add a piece of coding called "hr" which places a line border to separate the text. And VUALA!!! you got the format of my website!
