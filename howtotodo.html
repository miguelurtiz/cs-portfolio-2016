How To: To-Do List

Each "How To" Page will give me the chance to explain the different coding methods that I use when creating this website about me and my computer scinece experiences.

In this How To, I will explain how I used HTML, CSS, and JavaScript to code my very own properly working To-Do List. First, let's go over the...

HTML

With HTML, I am able to code elements and items to place o the website itself.


Now, in order to create a text box to insert you text, we would have to create a new "input" tag, followed with an id. The id will be named "input". Then, we would have to create a selection tag by typing "select id", and naming it "colorinput".




Next, we would have to create "input values" in order for the user to choose the text color they want. I specifically chose these colors because they represent the color of the Mexican flag, representing my ethnicity.




Then, we would have to create the button so that way, the user is able to submit their text to the space below the text box. In order to do this, we would need to add a "button id" named "Enter".




We just created the text box, the color selector, and the enter button. Now, we need to start coding with...

CSS

With CSS, I am able to code in order to change the look and the positions of the various elements scattered on the webpage.


Firstly, we need to make sure that the CSS file is connected to the HTML file. In order to do this, just simply add a "link" tag and place the CSS file name in the quotation marks.




In the CSS file, I can create properties in order to change the position and the look of any element I created on my HTML file.

NOTE* (If you would like to know more about the different properties and how they work, check out my How To for my Hardware page. That's you, Mr. Hesby).


Lastly, There is one more type of coding software that I would need to use in order to make the To-Do List function properly, and that would have to be...

JavaScript

The JavaScript is one of the most important coding softwares in my project. It acts like the brains for my To-Do List webpage.


Firstly, we need to make sure that the JavaScript file is connected to the HTML file. To do so, we must add a "script" tag in the HTML file, and type in the JavaScript file name within the quoation marks.





Each "var" property shown on the left is used to gather the names of the elements created in the HTML. they are searched by their ID and can be selected in the JavaScript file if they are titled under "var" properties.


By creating an ".addEventListener", and typing the function "click" next to it, we are able to make the "Enter button on the To-Do List clickable and functionable. The "var" property titled, "color", is where the JavaScript is atering information on whic color you chose to make the text with.



By creating a "ListItem" property, the JavaScript is able to produce the text that the user types and places it below the "Delete All" button.






Lastly, In order to make the "Delete All" button clickable and functional, we would need to create another new "var" property entitled, "remove". By adding a "removeChild" property, once the "Delete All" button is clicked, the entire website will refresh and so will the list produced below.
